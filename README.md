# Balance de Camioneta 🚚

Aplicación desarrollada en Python para calcular el balance financiero de una camioneta de acarreos.

## Descripción

Este programa permite calcular la rentabilidad de una camioneta dedicada a trabajos de acarreos, registrando ingresos y gastos para conocer las ganancias reales del vehículo.

## Funciones del programa

- Registro de acarreos
- Registro de gastos (gasolina, mantenimiento y otros)
- Cálculo de ganancias semanales, mensuales y anuales
- Cotización de acarreos por distancia
- Cálculo del valor por kilómetro
- Almacenamiento de datos en formato JSON
- Versión con interfaz de colores

## Archivos del proyecto

- *balance_camioneta.py* → primera versión del programa
- *camioneta_color.py* → versión con intento de interfaz en color
- *APK.py* → versión final con interfaz en color

## Tecnologías utilizadas

- Python 3

## Cómo usar el programa

1. Descargar el repositorio
2. Tener Python 3 instalado en el computador
3. Ejecutar el archivo principal

Ejemplo en la terminal:

python APK.py

El programa se ejecutará en la consola y permitirá ingresar datos para calcular ingresos, gastos y ganancias de la camioneta.
